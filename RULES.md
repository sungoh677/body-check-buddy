# Project Rules & Knowledge

## 핵심 가치 (Core Values)
1. **20초 점검 (20-Second Check)**: 모든 서비스 기능은 사용자가 20초 이내에 자신의 상태를 빠르고 직관적으로 점검할 수 있도록 설계한다.
2. **다크 모드 유지 (Maintain Dark Mode)**: 헬스케어 서비스다운 깔끔하고 전문적인 톤앤매너를 위해 현재 적용된 다크 모드 테마를 절대적으로 유지한다.

## 기술 및 개발 규칙
1. **Language**: 모든 답변과 코드 주석은 한국어로 작성한다.
2. **Tech Stack**: React, TypeScript, Tailwind CSS, Vite, Supabase를 기반으로 하며, any 타입 사용을 엄격히 금지한다.
3. **Architecture**: 코드는 가능한 한 역할별로 분리하고, 하나의 파일에 모든 로직을 넣지 않는다. (Clean Architecture 지향)
4. **Safety**: 중요한 파일(DB 설정 등)을 수정하거나 삭제하기 전에는 반드시 사용자에게 먼저 묻는다.
